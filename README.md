# Industry-Symbiosis: Resource Trading for Industrial Networks
A prototype platform developed for a university seminar, designed to help reduce resource waste efficiently.

The solution enables resource trading within an industrial network by repurposing production waste. Competitive advantages are created through the efficient trading of material and energy flows. This benefits members, the network, and the environment. There is currently no comparable software available. Typically, such trading within networks relies on personal discussions. The solution therefore stands out by providing APIs for automatic material and energy flow recording and display of input and output streams as well as matching between production facilities.
